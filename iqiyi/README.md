## 获取 cookie 教程

![获取 cookie 教程](./img/iqiyi_cookie.png)

1. 进入[爱奇艺（IQIYI）官网](https://www.iqiyi.com/)
2. 按 `F12` 打开开发者工具，刷新页面
3. 点击 `Network` 标签
4. 选择 `Doc` 标签
5. 选中 `www.iqiyi.com`
6. 下滑找到 `cookie` 全选复制即可

## 配置 config.json

将获取到的 cookie 写入 config.json
